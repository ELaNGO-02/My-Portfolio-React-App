# My-Portfolio-React-App

An interactive, visually-rich portfolio built with React, TypeScript, Three.js, and Tailwind CSS. This project showcases professional experience, projects, awards, and technical skills with 3D effects, animations, and a modern UI.

## 🚀 Features
- 3D animated backgrounds and interactive scenes (Three.js, @react-three/fiber)
- Modern, responsive design with Tailwind CSS
- Animated section transitions and effects (framer-motion)
- Project gallery with live demos and source code links
- Experience timeline, testimonials, and awards
- Custom animated cursor and code rain effects
- Contact form with EmailJS integration
- Fully mobile-friendly and accessible

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, clsx, class-variance-authority
- **3D & Animation:** Three.js, @react-three/fiber, @react-three/drei, framer-motion
- **UI/UX:** Radix UI, phosphor-react, lucide-react, react-icons
- **Utilities:** EmailJS, react-intersection-observer, react-parallax-tilt

## 📁 Folder Structure
```
src/
  components/      # UI, canvas, effects, layout, sections
  assets/          # Images, icons, 3D models
  constants/       # Config, data, styles
  hooks/           # Custom React hooks
  lib/             # Utility functions
  types/           # TypeScript types
  utils/           # Animation helpers
  globals.css      # Global styles
  App.tsx          # Main app component
  main.tsx         # Entry point
public/            # Static files, 3D assets
```

## 🖥️ Getting Started
1. **Install dependencies:**
	```bash
	npm install
	```
2. **Run the development server:**
	```bash
	npm run dev
	```
3. **Build for production:**
	```bash
	npm run build
	```
4. **Preview production build:**
	```bash
	npm run preview
	```
